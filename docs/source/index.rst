Welcome to stGP's documentation!
=================================

**stGP** is a spatiotemporal gene program modeling framework for spatial
transcriptomics. It decomposes target-cell gene expression across tissue
sections into latent gene programs, cell-level program activities, spatially
smooth residual fields, and temporal effects.

.. image:: figures/overview.png
   :align: center
   :alt: stGP model overview

This tutorial website provides runnable notebook examples for applying ``stGP``
to aging brain and injured kidney spatial transcriptomics studies. The source
code for building this website is intended to live at
https://github.com/Jamesyu420/stGP-tutorial.
